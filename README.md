# Explore With Me

Explore With Me — многомодульное приложение для публикации, поиска и участия в событиях.

Проект состоит из основного сервиса и сервиса статистики просмотров.

## Архитектура

Проект включает два микросервиса:

### ewm-main-service

Основной сервис приложения.

Функциональность:

- создание пользователей;
- создание и управление категориями;
- создание событий;
- публикация событий администраторами;
- поиск и фильтрация событий;
- подача заявок на участие;
- просмотр подробной информации о событиях.

Порт:

```
8080
```

---

### stats-server

Сервис статистики.

Функциональность:

- сохранение информации о просмотрах;
- подсчёт количества просмотров;
- предоставление статистики по URI.

Порт:

```
9090
```

## Сборка проекта

Выполнить из корневой директории:

```bash
mvn clean package
```

После успешной сборки будут созданы JAR-файлы:

```
ewm-main-service/target/ewm-main-service-0.0.1-SNAPSHOT.jar

stats/stats-server/target/stats-server-0.0.1-SNAPSHOT.jar
```

---

## Запуск через Docker

Сборка и запуск всех сервисов:

```bash
docker compose up --build
```

Запуск в фоне:

```bash
docker compose up --build -d
```

Остановка:

```bash
docker compose down
```

Полное удаление контейнеров и томов:

```bash
docker compose down -v
```

## Статистика просмотров

При обращении к публичным эндпоинтам событий:

```
GET /events
GET /events/{eventId}
```

основной сервис отправляет информацию о запросе в сервис статистики.

Статистика хранится отдельно в сервисе `stats-server` и используется для подсчёта просмотров событий.
